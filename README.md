# Algorithm Visualizer

A web-based tool to visualize core Data Structures and Algorithms.

## Algorithms Implemented
- Bubble Sort
- Merge Sort

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript

## How to Run
pip install flask  
python app.py

## Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)

## Tree Algorithms
- Inorder Traversal
- Preorder Traversal
- Postorder Traversal

## Pathfinding Algorithms
- A* (A Star) Pathfinding on Grid
